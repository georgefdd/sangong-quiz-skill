# 上海三公-小升初-真题-伴你学 📚

1.根据孩子当前学习阶段，生成三公小升初规划；
2.帮助孩子制定三公小升初真题检测规划（生成定时任务，定时推送真题，检测学习进度）；
3.根据孩子学习进度，学习情况，查漏补缺，针对性形成学习规划；

## 功能

- 为三年级学生提供每周真题练习
- 包含三校历年真题（2007-2025）
- 考点分析与备考建议
- 支持定时任务自动推送

## 适用场景

- 上海三公学校备考
- 小升初真题练习
- 题型规律总结

## 使用方法

```
# 提供孩子基本信息，制定真题测试规划
Anthony是9岁的上海三年级小学生
制定一个每周一测的定时任务

# 按学校选题
出一套上外附中的数学题

# 按难度筛选
出两道适合三年级的入门题
```

## 三校特点

| 学校 | 难度 | 特色 |
|------|------|------|
| 上外附中 | ⭐⭐⭐⭐⭐ | 英语+数学跨学科，新题型多 |
| 上实 | ⭐⭐⭐⭐⭐ | 面试实验设计，文言文数学题 |
| 浦外 | ⭐⭐⭐⭐ | 英语口语要求高，数学相对简单 |
