# 三公真题推送技能 📚

上海三公（上外附中、上实、浦外）小升初真题推送技能。

## 功能

- 为三年级学生提供每周真题练习
- 包含三校历年真题（2007-2025）
- 考点分析与备考建议
- 支持定时任务自动推送

## 适用场景

- 上海三公学校备考
- 小升初真题练习
- 题型规律总结

## 使用方法

```
# 推送本周练习
推送本周真题给Anthony

# 按学校选题
出一套上外附中的数学题

# 按难度筛选
出两道适合三年级的入门题
```

## 三校特点

| 学校 | 难度 | 特色 |
|------|------|------|
| 上外附中 | ⭐⭐⭐⭐⭐ | 英语+数学跨学科，新题型多 |
| 上实 | ⭐⭐⭐⭐⭐ | 面试实验设计，文言文数学题 |
| 浦外 | ⭐⭐⭐⭐ | 英语口语要求高，数学相对简单 |
