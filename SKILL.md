# 三公真题推送技能

为Anthony（9岁，三年级，青浦兰生复旦学校）提供上海三公学校小升初真题练习。

## 适用场景

- 推送真题练习
- 备考训练
- 题型分析
- 错题整理

---

## 🆕 P0优化：Harness架构增强

### 1️⃣ 题目索引（Question Indexing）

为提升选题效率，建立智能题库索引机制：

#### 索引结构
```yaml
题库索引:
  按学校分类:
    上外附中: [英语真题库, 数学真题库]
    上实: [数学真题库, 语文真题库]
    浦外: [英语真题库, 语文真题库]
  按难度分类:
    基础: 1-2星题目
    中等: 3星题目
    进阶: 4-5星题目
  按题型分类:
    选择题/填空题/解答题/口奥题
```

#### 索引更新流程
1. 每次出题后更新索引命中记录
2. 记录题目使用次数，避免重复
3. 标记高频考点与低频考点

#### 选题优化
- **去重机制**：同一题目7天内不重复出现
- **覆盖率检查**：确保每学期覆盖80%以上考点
- **自适应难度**：根据正确率动态调整题目难度

### 2️⃣ 审计日志（Audit Logging）

全程记录操作日志，确保可追溯：

#### 日志格式
```yaml
audit_log:
  timestamp: "2025-01-01T10:00:00+08:00"
  action: "generate_paper"
  user_id: "Anthony"
  params:
    school: "上外附中"
    subjects: ["英语", "数学"]
    difficulty: "中等"
  output:
    paper_id: "SG-2025-W01-001"
    questions_count: 10
    estimated_time: 45
  status: "success"
  version: "1.0.2"
```

#### 记录的操作类型
| 操作类型 | 描述 | 必填字段 |
|---------|------|---------|
| generate_paper | 生成试卷 | school, subjects, paper_id |
| review_answer | 批改答案 | paper_id, score |
| query_history | 查询历史 | user_id, date_range |
| update_index | 更新索引 | paper_id, questions_used |

#### 日志存储
- 本地日志文件：`./logs/audit_YYYYMMDD.json`
- 日志保留周期：90天
- 日志加密存储，防止篡改

### 3️⃣ 输出验证（Output Validation）

保障交付质量，输出前必须校验：

#### 验证清单
```yaml
pre_delivery_check:
  1. 格式校验:
     - A4排版格式正确
     - 题号连续无跳号
     - 答案与题目匹配
  
  2. 内容校验:
     - 题目数量符合要求（8-12题）
     - 分值总和为100分
     - 无错别字、病句
  
  3. 质量校验:
     - 难度与设置匹配
     - 无超纲内容（针对三年级）
     - 答案解析准确
  
  4. 完整性校验:
     - 包含答案区域
     - 包含解析区域
     - 包含答题卡区域
```

#### 验证流程
```
选题 → 生成题目 → 自动校验 → 如有问题自动修复 → 再次校验 → 输出交付
```

#### 异常处理
- 校验失败自动重试（最多3次）
- 重试失败记录错误日志并告警
- 保留失败样本供人工复核

## 三校特点速查

### 上外附中
- **难度**：⭐⭐⭐⭐⭐
- **特色**：英语+数学跨学科，新题型多
- **时间压力**：每页独立倒计时，不可回翻
- **英语重点**：科普词汇、AI话题、图灵测试
- **数学重点**：数论、组合、几何、行程

### 上实
- **难度**：⭐⭐⭐⭐⭐
- **特色**：面试实验设计、文言文数学题
- **数学重点**：几何（圆与扇形）、行程、创新思维
- **语文重点**：文学常识、小作文（家事国事天下事）
- **面试**：设计实验（保温杯、死海浮力）

### 浦外
- **难度**：⭐⭐⭐⭐
- **特色**：英语口语要求高，数学相对简单
- **英语重点**：quick response、中译英、完形填空
- **数学重点**：口奥、逻辑推理、行程
- **语文重点**：古诗积累、文综常识

## 注意事项

1. **时间压力训练**：提醒限时完成，模拟真实考试环境
2. **错题记录**：每次练习后记录错题，建立错题本
3. **循序渐进**：从中等难度开始，逐步提升
4. **正向激励**：注重鼓励，建立信心
